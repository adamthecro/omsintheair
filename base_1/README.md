# OmsInTheAir Base 1

The primary base for OmsInTheAir helps us interact with the autonomous functions of the plane.

## Installation

Use the package manager [npm](https://www.npmjs.com/get-npm) to install the packages needed.

```bash
npm install
```

## Usage

```bash
npm start
```
