## Install

For the base programs, use the package manager [npm](https://www.npmjs.com/get-npm) to install the packages needed.

```bash
npm install
```
For the airplane software:

```bash
c++ main.cpp -o main -pthread -lcurl
```

## Usage

Base:

```bash
npm start
```

Airplane:

```bash
./main
```
